= 인타운 협력 (하이텍정보시스템)
= 도장전 용접가능
= 자동차는 용접부터 가능
= 용접 - 도장 - 마지막
= 용접조건이 중요 / 이것에 따라 불량이냐 아니냐
= 용접공정에서 작업자의 오류 행위를 막기위해 수집된 요건에 맞는지를 컨트롤 (자동화)
= 취합된 데이터를 통해 통계분석까지

= 기준치에 맞게 용접해야 내구성이 높아짐

＊용접관련
- 자동차는 봉으로 (이외로 스파크)
- 와이어로 된 봉으로 용접
- 봉은 가스가 없음
- 용접되는부분 : 용접팁
- 용접방식 (방법)
- 용접조건 : 전압, 전류, 가스량(아크용접), 냉각수온도 , 팁 교환주기, 
	   팁 드레싱(청소)주기 == 100번 드레싱하고 교체하라

* 고려할 부분
- 라인에 들어오는 전류 체크 떨어지면 알람
- 팁 교환주기 = 타점(용접한포인트)
- 부수적인 환경요인 = 냉각수 온도 (환경온도)
- 자동화에서 에어가 중요
- 용접할때 그외 환경조건 생각( 일반조건, 환경조건 ) 
- 통계분석관리시스템
- 기준치에 맞게 품질을 관리 (스펙 벗어나면 알림)
- 데이터 기반으로 수집된 데이터에 정상적인지 아닌지를 구분 == SPC
